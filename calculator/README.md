# A Simple Calculator
Calculator built with HTML, CSS and JavaScript that also tracks calculation history.

# Key Features
Performs basic arithmetic operations (+, -, *, /)
Saves every calculation as an object in an array.
Dectects division by zero error and returns null.

# How to Use
Open the file 'index.html' in a browser.
Enter two numbers in the different input spaces provided
View the result and the history of calculations below the input spaces.
